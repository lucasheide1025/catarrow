# 🎯 貓小隊射箭場 — 射手使用說明

---

## 登入

開啟網站後，輸入教練給你的帳號與密碼登入。  
登入後進入射手介面，底部有五個分頁。

---

## 底部導覽

| 圖示 | 名稱 | 功能 |
|------|------|------|
| 🏠 | 首頁 | 通知、最新消息、快捷入口 |
| 🏆 | 比賽 | 查看比賽、報名、記分 |
| 🎯 | 練習 | 自填練習記錄 |
| 📊 | 排行 | 全員積分排行榜 |
| 👤 | 我的 | 個人資料與所有功能入口 |

---

## 首頁

- 查看教練發布的最新通知
- 快速進入常用功能
- 如果你有**組隊進行中**，頂部會出現提示橫幅，點一下可以回到房間

---

## 比賽分頁

### 查看比賽
- 顯示所有進行中、即將開始的比賽
- 點入比賽可以看：比賽資訊、靶紙圖、報名名單、成績排行

### 報名
- 點擊比賽 → 點「報名」按鈕

### 記分
- 已報名 + 比賽狀態為「開放中」才能記分
- 點「開始記分」後逐回合輸入箭數環數
- 年度檢定需等教練審核後才算正式成績

### 打怪模式 🐉
- 在比賽分頁可進入打怪模式
- 選擇難度（根據你的積分等級解鎖不同怪物）
- 打怪後可能掉落素材，素材可用於合成或升級

### 決鬥模式 ⚔️
- 與其他射手 1v1 或組隊對戰
- 建立房間並分享房間號碼給對手
- 支援不對等人數對戰（1v2 等）
- 如果中途離開，頂部橫幅可以回到戰場

---

## 練習分頁

- 記錄每次自主練習的環數、箭數、心得
- 練習記錄會影響積分與成就進度

---

## 排行分頁

- 查看全體射手的積分排名
- 積分來源：比賽成績、徽章、打怪、成就等

---

## 我的分頁

點入「我的」後可以看到個人卡片，以及以下功能入口：

### 基本功能
| 功能 | 說明 |
|------|------|
| 學習記錄 | 教練幫你填寫的上課記錄 |
| 訊息 | 與教練的私訊對話 |
| 歷史成績 | 所有比賽的成績記錄 |
| 外部比賽 | 登記道館外參加的比賽 |
| 通知中心 | 所有系統通知 |

### 遊戲化功能
| 功能 | 說明 |
|------|------|
| 成就 | 查看已解鎖與未解鎖的成就 |
| 成就圖鑑 | 詳細成就列表與進度條 |
| 背包素材 | 打怪掉落的素材、道具 |
| 卡牌收集 | 收集的怪物卡牌 |
| 怪物圖鑑 | 打過的怪物記錄 |
| 組隊 | 建立或加入組隊房間（打怪任務或組隊戰鬥） |

### 年度檢定
- 在「年度檢定」頁面申請參加檢定
- 需教練審核才算通過，通過後會取得對應等級

### 主題切換
- 在「我的」頁面可以切換 App 的配色主題

---

## 組隊系統 🎮

1. 進入「我的」→「組隊」
2. 建立房間（選擇任務類型）或輸入房間號碼加入
3. 房間建立後等隊友加入
4. 開始任務或戰鬥
5. 組隊進行中時，頁面頂部會出現橫幅提醒

---

## 決鬥系統 ⚔️

1. 進入「比賽」分頁 → 點「決鬥模式」
2. 建立房間，將房間號碼告知對手
3. 對手輸入號碼加入後自動開始
4. 戰鬥採回合制，依雙方攻防計算傷害
5. 決鬥成績會記錄在成就與圖鑑中

---

## 常見問題

**Q：積分怎麼算？**  
A：比賽成績、徽章獎勵、打怪、成就解鎖都會影響積分，教練可以查到詳細明細。

**Q：記分記錯了怎麼辦？**  
A：聯絡教練請他幫你修改。

**Q：年度檢定要怎麼申請？**  
A：進入「比賽」分頁，找到年度檢定活動，點進去後選擇弓種開始記分，提交後等教練審核。

**Q：組隊房間號碼在哪找？**  
A：建立房間後畫面上會顯示，截圖傳給隊友即可。
