# 官網手機會員登入入口

## Goal

讓官網手機版使用者除了「立即預約」，也能直接找到並進入既有會員登入／會員中心。

## Requirements

- 手機導覽或主要行動區同時顯示「立即預約」與「會員登入」。
- 會員入口沿用既有 `?login=1` 公開預約登入流程；立即預約維持一般公開預約入口。
- 桌面版既有兩個入口不得退化。
- 入口文字、視覺層級與點擊區適合手機，會員登入不得藏在需猜測的選單內。
- 先定位 `archery.catgroup.com.tw` 對應原始專案與部署設定；不得在錯誤 Vercel 專案修改。

## Acceptance Criteria

- [x] 360px 寬手機首頁可直接看到並點擊會員登入。
- [x] 會員登入導向公開預約頁的登入狀態，立即預約導向選課狀態。
- [x] 桌面與手機導覽皆保留兩個入口。
- [ ] 官網建置與正式網址驗證通過。
