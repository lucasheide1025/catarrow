# 背包與金幣商店重建

> 建立日期：2026-07-04
> 狀態：實作中

## 問題陳述

目前「我的背包」同時扮演功能入口、物品倉庫、合成台與收藏頁，玩家無法快速理解哪些是持有物品、哪些是製作功能。金幣商店則在基本裝備買齊後缺少持續用途，商品選擇與後期經濟也沒有形成循環。

## 已確認事實

- `MemberInventoryHub` 目前混合金幣商店、材料、貓咪、故事、裝備、怪物卡片與貓貓村等入口。
- `MemberMaterials` 同時包含怪物素材、寶箱、徽章碎片與藥水製作。
- 背包藥水文案仍描述舊制「戰前攜帶三瓶、自動生效」。
- 現行戰鬥實作是每回合最多使用一次藥劑；投擲型會占用一箭。
- 藥水製作已存在於貓貓村，且後端實際消耗村莊資源與金幣。
- 背包舊藥水製作畫面卻以怪物材料庫存判定是否可製作，會產生錯誤提示。
- 金幣商店裝備在十個槽位買齊後失去主要用途。
- 裝備品牌只影響外觀，但目前在裝備頁可直接免費切換，與商店定位重疊。
- 商店消耗品是固定五項硬編碼商品，材料包只涵蓋部分怪物族群。
- 現有購買流程先扣金幣、後發商品，不是單次原子交易；發放失敗可能造成玩家損失。

## 產品原則

1. 背包只呈現玩家實際持有的物品與對這些物品的直接操作。
2. 製作功能回到其世界觀設施；藥水只能在貓貓村製作。
3. 收藏、角色養成與商店不偽裝成背包分類。
4. 金幣商店提供持續但不直接破壞戰鬥成長曲線的用途。
5. 所有購買必須保證扣款與發放的一致性，並防止重複點擊或重試造成重複交易。

## 背包需求

### 分類

- 戰利品：材料箱、金幣箱、藥水箱、卡包與其他未開啟容器。
- 消耗品：回復、能力強化、投擲與控制類藥劑。
- 怪物素材：打怪取得及裝備強化使用的材料。
- 徽章碎片：顯示持有量、合成需求與可合成狀態。
- 特殊道具：保留活動券、任務道具等擴充分類。

### 介面與操作

- 預設只顯示已持有物品，不把未解鎖圖鑑混入背包。
- 每件物品以雙欄小卡片顯示圖示、名稱、稀有度與數量。
- 點擊卡片可查看效果、用途、來源及可用位置。
- 戰利品保留單箱開啟與全部開啟。
- 怪物素材保留現有五換一升級。
- 徽章碎片保留合成功能。
- 移除背包內的藥水製作清單，改為導向貓貓村藥水工坊。
- 可操作提示只用於有箱子可開、有材料可升級或有碎片可合成等實際狀態。

## 金幣商店需求

### 分區

- 今日精選：每天全服同步更新一般補給與素材，購買限制按玩家獨立計算。
- 每週珍寶櫃：每週全服同步更新一件稀有商品，每位玩家該週限購一次。
- 戰鬥補給：寶箱、指定族材料包與基礎消耗品。
- 外觀收藏：永久解鎖裝備品牌或其他不影響能力的外觀。
- 素材回收：每日限量將多餘低階素材換回金幣。

### 裝備與外觀

- 尚未取得的基本裝備槽位可繼續透過商店補齊。
- 已取得的槽位不重複販售能力值。
- 裝備品牌改為永久解鎖；解鎖後可在「我的裝備」自由切換。
- 商店不直接販售裝備品級或強化等級。

### 商品資訊

- 商品卡顯示玩家目前持有數量、價格、用途與剩餘購買次數。
- 購買成功後明確指出商品進入哪個背包分類。
- 材料包需涵蓋所有怪物族群，不再硬編碼少數族群。
- 商品及價格應集中定義，避免 UI 與資料層各自持有不同規格。
- 今日精選採全服相同商品，所有玩家在同一天看到相同內容。
- 今日精選的購買次數由每位玩家獨立計算，不共享全服庫存。
- 今日精選平時由系統自動輪替，管理員可設定指定日期的覆蓋商品。
- 所有自動與手動商品都必須取自經確認的可販售白名單；珍貴、活動限定或會破壞進度的道具不得因共用資料表而自動上架。
- 一般稀有商品應讓活躍玩家在數天內負擔；咪咪箱、地下城卷軸等核心珍寶定位為約一至兩週的中期存錢目標，不設計成數週至數月的長期門檻。
- 商品價格全服固定，不因玩家等級或既有資產而浮動。

### 候選商品池（待逐類確認）

#### 常態／自動輪替候選

- 鐵寶箱、黃金寶箱、藥水箱。
- 六族 T1～T3 指定素材包。
- 普通與非凡級消耗品。
- 尚未取得的基本裝備槽位。
- 已確認可販售的純外觀解鎖。

#### 稀有輪替候選

- 史詩寶箱。
- 怪物卡包。
- 咪咪箱。
- 世界王地下城卷軸。
- 稀有級藥水或指定素材選擇箱。
- 少量扭蛋幣；每次少量且採個人限購。

以上稀有候選放入「每週珍寶櫃」，不混入每日一般商品輪替。

#### 管理員限定上架候選

- 神話寶箱。
- 貓貓箱。
- 高階 T4～T6 素材選擇箱。
- 活動限定外觀。

#### 禁止販售候選

- 地下城首通限定收藏品與一般地下城收藏品。
- 成就、稱號、認證、徽章成品及冒險者階級。
- 實體獎品與折價券。
- 世界王擊殺／最後一擊等證明型獎勵。
- 任何可直接提高裝備品級、強化值或角色等級的商品。
- 箭露；箭露同時用於村莊升級與玩家卡片市場，不建立金幣兌換匯率。

## 初步驗收標準

- [ ] 背包主畫面不再顯示商店、故事、貓咪、村莊等非物品分類。
- [ ] 背包不再出現舊制藥水說明或重複的藥水製作按鈕。
- [ ] 持有藥水的數量與單人／組隊戰鬥實際庫存一致。
- [ ] 所有背包分類都有空狀態、持有數與可操作狀態。
- [ ] 怪物素材升級、徽章合成、單箱及全部開箱仍可正常使用。
- [ ] 商店在基本裝備買齊後仍有今日精選、補給、外觀及回收用途。
- [ ] 裝備品牌購買一次後永久解鎖，不會重複收費。
- [ ] 商店購買與素材回收具備交易一致性及重複操作防護。
- [ ] 正式 build 通過。

## 不在本次範圍

- 重新設計戰鬥中的藥劑效果和平衡數值。
- 將所有既有 Firestore 庫存集合強制搬入單一新集合。
- 販售直接提高裝備品級、強化值或角色能力的付費商品。

## 已核准定價方向

- 每日補給：150～800 金幣。
- 一般珍寶：1,000～1,800 金幣。
- 咪咪箱：2,500 金幣。
- 地下城卷軸：3,000 金幣。
- 管理員限定珍品：3,000～5,000 金幣。
- 價格由集中設定管理，不寫死於畫面元件。
