# 訪客預約體驗重整

## Goal

重整訪客從官網進入、查看時段、註冊／登入到完成預約的手機體驗，使資訊清楚、操作安全且符合貓小隊射箭場的高質感品牌形象。

## Deliverables

- `07-16-public-booking-ui`：訪客預約頁的可預約狀態、日期範圍、時數呈現與整體響應式改版。
- `07-16-website-mobile-member-login`：官網手機導覽補上會員中心／登入入口。

## Acceptance Criteria

- [ ] 訪客不會看到任何新生／舊生或實際人數統計，只看到能否預約。
- [ ] 訪客只能瀏覽今天至一個月內的日期，不能往前。
- [ ] 所有預約確認與課程卡片都能直接看出開始、結束與總時數。
- [ ] 手機使用者能從官網直接進入會員登入／中心，不只看到立即預約。
- [ ] 訪客註冊與預約流程完成手機優先、高質感且可存取的視覺重整。

## Boundaries

- 不改預約容量交易、價格、營業時間或 Email 通知邏輯。
- 官網與學生系統若屬不同專案，必須分開建置與部署，不以複製靜態碼替代。
