# Design：官網 SEO/GEO 內容實作

## 1. 檔案結構與模板複製法

`website/` 沒有 build 流程、沒有 include/樣板系統，純手寫靜態 HTML，且部署是把 `website/` 內容整包複製到另一個獨立 Vercel 專案（不能依賴這個 repo 根目錄的 `vercel.json`）。因此：

- 7 支新頁面各自建立資料夾＋`index.html`：`website/rainy-day/index.html`、`website/sunny-day/index.html`、`website/beginner-guide/index.html`、`website/family/index.html`、`website/couple/index.html`、`website/friends-group/index.html`、`website/corporate-team-building/index.html`、`website/solo-friendly/index.html`（共 7 個資料夾）。這樣部署後網址自然是 `archery.catgroup.com.tw/rainy-day/` 這種乾淨形式，不需要任何伺服器端 rewrite 設定。
- 每個新頁面的 `<head>`（含 `<style>` 整塊）、`<header id="hd">` 導覽列、`<footer>`，直接從 `website/index.html` **複製**過去，只改：
  - `<title>`／`<meta name="description">`／canonical URL／OG 標籤裡的 URL 與描述
  - 該頁自己的 LocalBusiness schema **不要複製**（只有首頁該有），改成只放該頁專屬的 FAQPage schema
  - header 導覽列裡的錨點連結（`#why`／`#bows` 等）在子頁面要改成 `/#why`／`/#bows`（加上回首頁的路徑前綴），因為子頁面沒有這些 section
  - `<main>` 內只放這一頁自己的內容（不是整份首頁內容）
  - 相對路徑的資源（`assets/xxx.png`、CSS 變數等）因為子頁面多了一層資料夾，圖片路徑要改成 `../assets/xxx.png`
- 圖片素材：優先重複使用 `website/assets/` 現有的貓咪插畫/照片（不需要為每頁生新素材），依主題挑選語意相近的既有圖（例如親子頁可用團體照 `assets/009.png`，情侶頁/I人頁可用 `assets/006.png` 戴耳機專注拉弓的插畫）。

## 2. Schema 配置

- **LocalBusiness/SportsActivityLocation**：維持只在首頁 `index.html` 一份，7 支新頁不重複宣告。
- **FAQPage**：
  - 首頁：在既有 8 題 JSON-LD 陣列後面**追加** C 段的 10 題（見下方第 4 節逐字內容），維持同一個 `FAQPage` 物件、同一個 `mainEntity` 陣列。
  - 每支新頁面：各自帶一份**獨立**的 `FAQPage` schema，只放該頁專屬的 3-4 題（見第 5 節逐頁規格裡的 FAQ 內容），不要跟首頁的 10+8 題重複文字（重複文字的 FAQ schema 在多頁面之間會被視為重複內容，稀釋 SEO 價值）。

## 4. C 段：首頁新增 FAQ（10 題，逐字內容，直接採用）

按順序追加到現有 FAQPage schema 的 `mainEntity` 陣列尾端，並在 `.faq-list` 視覺清單同步新增對應的 `<details>`：

1. **Q: 台南下雨天可以來射箭嗎？** A: 可以，而且很適合。貓小隊是室內射箭場，不受下雨或午後雷陣雨影響，一樣正常開放，是雨天很實用的活動備案。
2. **Q: 台南颱風天可以預約嗎？** A: 若遇颱風警報、停班停課或交通狀況不佳，請以安全為優先，不建議冒險出門；一般天氣不穩、午後陣雨或炎熱天氣時，室內射箭才是合適的活動備案。實際營運狀況請以現場公告為準。
3. **Q: 台南大太陽的天氣適合室內射箭嗎？** A: 很適合。場地全程室內有空調，不用擔心曬傷或流汗狼狽，是夏天午後想找地方避暑的好選擇。
4. **Q: 小朋友可以射箭嗎？幾歲開始比較適合？** A: 滿 5 歲以上即可參加，現場會依年齡、身高與臂力安排適合磅數的器材。並非所有年齡都適合同一種弓，實際狀況以現場教練評估為主。
5. **Q: 情侶約會適合來射箭嗎？** A: 適合。比起單純吃飯看電影，射箭多了互動與挑戰——可以互相比成績、一起拍照，是個有記憶點的約會行程。
6. **Q: 一個人來射箭會不會很尷尬？** A: 不會。單人體驗完全不需要組隊或社交，教練會協助安排，很多客人都是自己一個人來，安靜地專注在每一次放箭上。
7. **Q: 完全沒射過箭，第一次可以直接參加嗎？** A: 可以。貓小隊主打新手友善，教練會從安全規則、持弓、站姿、開弓到瞄準逐步引導，不綁約、不推銷課程。
8. **Q: 朋友聚會或慶生活動適合安排射箭嗎？** A: 適合，小團體也能參與，不需要運動背景。可以安排彼此比分數、挑戰靶心，很適合當作慶生或聚會的活動之一。
9. **Q: 射箭適合安排公司團康或部門活動嗎？** A: 適合。射箭不要求體能或運動背景，可以安排個人挑戰或分組競賽，適合想找有互動、但不過度激烈的室內團康活動。
10. **Q: 下班後想放鬆一下，射箭適合嗎？** A: 很適合。射箭的節奏偏安靜，重點在專注呼吸、站姿與每一次放箭，很多人拿來當作下班後轉換心情的活動，但並非醫療用途，僅供休閒參考。

## 6. 內容風格檢查（實作時逐頁比對）

- 颱風天：不能寫成鼓勵危險出門，必須是「警報/停班停課/交通不佳以安全優先，一般天氣不穩才是室內活動的合適備案」這種表達（已在 C 段第2題示範）。
- 兒童年齡：不能寫「任何年齡都適合」，要寫「滿5歲以上，現場依年齡身高安排器材」。
- 紓壓/專注：不能宣稱治療效果，只能講「休閒/轉換心情用途」。
- 每段落 2-4 行，手機閱讀為主，FAQ答案第一句話要能獨立回答問題（AI摘要常只截取前1-2句）。
