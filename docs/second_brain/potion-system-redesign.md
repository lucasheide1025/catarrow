# 🧪 Potion System Redesign — 藥水系統大改版規格
> 最後更新：2026-06-28
> 狀態：設計定稿，待實作

---

## 一、改版背景

原有藥水系統有兩個獨立系統：
1. **戰鬥藥水**（`itemData.js`）：8 種藥水，只能在打怪/組隊戰前選 3 瓶喝，效果只有一場
2. **地下城商店**（`dungeonData.js`）：8 種商品，只能在地下城商店用金幣買，整局有效

**問題**：
- 兩個系統各自獨立，零交集
- 背包累積的藥水在地下城完全用不到
- 戰鬥藥水效果太複雜（爆擊率、穿心+1分、麻痺毒素等）
- 地下城商店的 buff 太單調

**解決方案**：三層藥水系統 + 底部 tab 列 UI 改造

## 二、三層藥水架構

### 分層總覽

| 分類 | 原名稱 | 範圍 | 效果 | 使用方式 |
|------|--------|------|------|---------|
| **❶ 攜帶型藥水** | 舊型藥水系統 (itemData.js) | **全模式通用**（打怪/組隊/地城） | 簡單 buff（回HP、+ATK、+DEF） | 回合中從藥水 tab 點選，消耗一瓶 |
| **❷ 投擲型道具** | 麻痺毒素、投擲毒藥等 | **全模式通用** | 戰鬥中丟怪：扣HP、降ATK、降DEF、麻痺、小刀 | 取代一箭，消耗一瓶 |
| **❸ 地下城商店** | dungeonData.js | **只有地下城能用** | 維持現有商品不變 | 金幣買，整局有效 |

### ❶ 攜帶型藥水（全模式通用，消耗品）

| 藥水 | 效果 | Lv1 | Lv2 | Lv3 | 製作材料 |
|------|------|-----|-----|-----|---------|
| ❤️ HP恢復 | 回 % 血量 | 5% | 10% | 15% | 貓草藥水 T1/T2/T3 + 金幣 |
| ⚔️ ATK提升 | 本回合 ATK +% | +5% | +10% | +15% | 貓草藥水 T1/T2/T3 + 金幣 |
| 🛡️ DEF提升 | 本回合 DEF +% | +5% | +10% | +15% | 貓草藥水 T1/T2/T3 + 金幣 |

**使用規則**：
- 每回合只能選 **一種** 攜帶型藥水使用
- 使用後 **藥水頁籤灰掉**（本回合不可再開），自動退回計分 tab
- 下回合解鎖，可再次使用
- 消耗一瓶庫存

### ❷ 投擲型道具（全模式通用，消耗品）

| 道具 | 效果 | 製作材料 |
|------|------|---------|
| 💉 **固定傷藥水** | 對怪固定扣 30 HP | 村莊材料 T2 + 金幣 |
| 💉 **比例傷藥水** | 對怪扣 maxHP 10% | 村莊材料 T3 + 金幣 |
| 💉 **隨機傷藥水** | 對怪扣 15~50 HP | 村莊材料 T2 + 金幣 |
| 🧪 **降ATK藥水** | 怪物本場 ATK-20% | 村莊材料 T3 + 金幣 |
| 🧴 **降DEF藥水** | 怪物本場 DEF-20% | 村莊材料 T3 + 金幣 |
| 🕸️ **麻痺藥水** | 禁止怪物反擊一次（全隊共用，觸發即消） | 村莊材料 T4 + 金幣 |
| 🔪 **投擲小刀** | 直接傷害，不吃 ATK/DEF（固定 15 傷） | 村莊材料 T2 + 金幣 |

**使用規則**：
- 點了取代一箭 → 箭格顯示 💉/🔪 圖示
- 可與分數混合使用（如 🔪[X][9][8][7][6]）
- 使用後 **藥水頁籤灰掉**（本回合不可再開），自動退回計分 tab

### ❸ 地下城商店（維持不變）

維持 `dungeonData.js` 現有商品，用金幣購買，整局有效。

## 三、舊系統對照表

### 被取代的舊藥水

| 舊藥水（itemData.js） | 新系統 |
|----------------------|--------|
| 小生命藥水 (HP+15%) | → HP恢復 Lv3 (15%) |
| 大生命藥水 (HP+30%) | ❌ 移除（效果過強，不符合簡單設計） |
| 力量藥水 (ATK+15%) | → ATK提升 Lv3 (15%) |
| 狂暴藥水 (ATK+30%) | ❌ 移除 |
| 爆擊靈藥 (crit+20%) | ❌ 移除 |
| 穿心藥水 (每箭+1分) | ❌ 移除 |
| 投擲毒藥 (monATK-30%) | → 降ATK藥水 (ATK-20%) |
| 麻痺毒素 (15傷+跳過反擊) | → 🕸️ 麻痺藥水（純麻痺）+ 💉 固定傷藥水（純傷害） |
| 神聖水 (ATK+15%+DEF-25%) | ❌ 移除 |
| **新增** | 💉 比例傷藥水、💉 隨機傷藥水、🔪 投擲小刀 |

### 被取代的舊配方

```
舊：怪物材料（ghost_m1, mountain_m1...）→ 製作藥水
新：怪物材料 → 只做裝備升級
    貓草藥水（村莊資源 potion_t1~t3）→ 製作攜帶型藥水
    村莊材料（礦物/瓜瓜/鮮魚...）→ 製作投擲型道具
```

### 製作原則

```

  怪物材料 → 只做裝備升級（ＲＰＧ裝備）
```
